Selamat Datang di Kantin Enca

Ikuti langkah mudah untuk memesan makanan favorit Anda!

Langkah 1: Tekan tombol E-Menu

Dengan adanya E-Menu membuat pembeli dapat mengetahui apa saja yang tersedia.

Langkah 2: Tekan Tombol "+" untuk memilih pesanan

Untuk memilih pesanan apa yang akan dipesan, pembeli dapat menekan tombol "+" untuk memilih berapa banyak pesanan dan tombol "-" untuk mengurangi pesanan.

Langkah 3: Isi data pembeli

Isi data singkat seperti nama pemesan, catatan pembeli, dan cara pengantaran.

Langkah 4: Tekan tombol "Pesan Sekarang"

Periksa kembali pesanan Anda, lalu klik "Pesan Sekarang" untuk mengonfirmasi dan mengirimkan pesanan.

Langkah 5: Tekan tombol "Kirim via WhatsApp"

Setelah pesanan berhasil dibuat, maka struk pembelian akan dicetak, dan pembeli dapat menekan langsung tombol "Kirim via WhatsApp".

Langkah 6: Kirim Struk ke penjual melalui WhatsApp

struk otomatis terkirim di draf pesan dan pembeli tinggal menekan tombol enter untuk mengirim pesanan.
